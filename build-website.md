Use Next.js to build a static website with a pipeline that runs a local webserver and set up github actions to push all files that you built on commit to a FTP server. 

Make sure you download all custom fonts, espacially if you use google fonts or bootstrap.
In this repo you can already find all the static content you need (see ./static)

You can get info on who we are and what we do from the manifest file. Use our Logo.

The website is supposed to contain our manifest, contact details, short summary who we are and what we do, template for link tree.
